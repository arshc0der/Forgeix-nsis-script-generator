# 📌 Overview

**Forgeix** is a modern desktop application that visually generates **NSIS installer scripts**, eliminating the need to manually write complex `.nsi` files.

Whether you're an indie developer, software company, or student, Forgeix helps you build professional Windows installer scripts through a beautiful and intuitive interface.

Designed with a premium **Apple-inspired glass UI**, Forgeix focuses on simplicity, speed, and developer productivity.

# 🛠 Installation

## Windows

1. Download the latest release from the **Releases** page.

2. Run the installer.

3. Launch **Forgeix**.

---

## Build From Source

```bash
git clone https://github.com/arshc0der/Forgeix-nsis-script-generator.git

cd Forgeix-nsis-script-generator
```

Open the project using **Qt Creator** and build using **Qt 6**.
